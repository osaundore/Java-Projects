 # ATM Management System (Java)

## Project Description
This is a console-based ATM Management System developed using Java and executed through the Command Prompt (CMD).  
The project simulates basic ATM operations such as checking balance, depositing money, withdrawing money, and exiting the system.

## Technologies Used
- Java (JDK)
- Command Prompt (CMD)

## Features
- Check account balance
- Deposit money
- Withdraw money
- Exit the application
- Console-based interface
